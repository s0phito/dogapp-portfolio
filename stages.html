Stage 1

The deliverables for this stage included:

  • Forming a team of 5
  • Completing a team contract
  • Establishing an online repository
  • Creating an online project portfolio
  • Coming up with 3 project ideas
The contract and team formation were all done within the first tutorial. We decided to use GitHub for our online repository. The repository is currently private and thus, cannot be reached at the moment but we will be pushing all things related to our project onto the repository. The online project portfolio was created using html5 & css3 and is hosted using GitHub pages.
Here are the 3 ideas submitted:
  • Online Internship Search Engine
  • Schedule Builder App
  • Informational App for Dog Owners/Lovers
